# Roman Marach Portfolio (Vite + React + Tailwind)

Modernized portfolio site migrated from Create React App to Vite.

## Tech
- Vite
- React 18
- React Router 6
- Tailwind CSS
- Framer Motion

## Run locally
1. `npm install`
2. `npm run dev`
3. Open the local URL shown in terminal

## Build preview
1. `npm run build`
2. `npm run preview`
